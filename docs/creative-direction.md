# DevClub Concurso — Creative Direction

## Concept

### NEXT LEVEL

The DevClub experience is designed as an interactive journey.

The user moves through different stages:

1. Curiosity
2. Challenge
3. Learning
4. Building
5. Evolution
6. Career
7. Next Level

## Visual Direction

The visual identity combines:

- Dark Tech
- Cinematic
- Editorial
- Interactive

The website should feel like an interactive digital experience rather than a traditional institutional landing page.

## Main Message

> You are not just learning to code.
> You are building your next level.

## Main Evaluation Priorities

The project is optimized around the official evaluation criteria:

- 50% — Visual impact and originality
- 30% — Animations and microinteractions
- 20% — Code quality and organization

## Experience

The user should experience the website as a journey:

Curiosity → Challenge → Learning → Building → Evolution → Career → Next Level

## Technical Direction

The project uses:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion
- Lenis
- Lucide React

The architecture should prioritize:

- Componentization
- Reusability
- Maintainability
- Performance
- Accessibility
- Responsive design
